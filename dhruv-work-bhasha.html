Bhasha Translate

A responsive, lightweight desktop application that translates text between English and Hindi in both directions. Designed with a clean, distraction-free layout and a focus on smooth, non-blocking background execution.

My Role & Contributions

  • Asynchronous Processing – Integrated Python's threading module to execute Google Translate requests in a background thread, preventing UI freezing or "Not Responding" states.
  • Smart Direction Swapping – Programmed a bidirectional swap mechanism that reverses the translation direction (English ↔ Hindi) and automatically exchanges text content between the input and output boxes.
  • Desktop GUI Implementation – Designed and configured a resizable Tkinter window with scrollable source/output fields, custom button palettes, and responsive layouts.
  • Robust Error & Length Handling – Built friendly network connection warning dialogs using Tkinter's messagebox and implemented length validation for inputs up to 5,000 characters.

Tech Stack

Python, Tkinter (GUI), deep-translator (GoogleTranslator API Wrapper), Threading.

Outcome

Delivered a polished and responsive translation utility for Windows/macOS that runs efficiently, consumes minimal system resources, and remains perfectly fluid during network delays.
